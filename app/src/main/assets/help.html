SshClient, version

Outer World Apps

A basic SSH Client that can be used to access remote computers that are accessible via SSH. Unlike some other Android SSH Clients, this one only requires internet access permission to make SSH connections and write external storage permission for the file transfer functions.

There are six screens:

  • SHELL - contains the currently selected session's contents, consisting of a connect-to single-line text box and a large screen text box. This is the screen you are usually on when communicating with a remote host.
  • FILE TRANSFER - lists the directory you are currently pointed to, either on the remote host or on the local device.
  • TUNNELS - lists the defined tunnels for the current session and provides a method to define new ones, and open, close, delete existing ones.
  • SETTINGS - this has various settings that you can adjust as you like.
  • VNC CLIENT - accesses a VNC server located on a server connected to by the current session.
  • HELP - this screen.
You can navigate to the desired screen by clicking the Android's menu button then clicking More and then selecting the screen you want.
